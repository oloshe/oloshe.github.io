新拟态风格css代码_紧跟潮流

发表于 2020-06-05

简介

新拟态是一种图片样式（风格），其原理是为界面的UI元素赋予真实感。

不同于扁平风的平面感和投影风的漂浮感，新拟态介于两者之间，能感受到他的“厚度”，但没有漂浮感，像是凸起的感觉。

他的特点主要是背景颜色与按钮颜色一致，依赖于左下角和深色阴影和右上角的高光阴影形成的一种厚度感。
